Screwdriver is an Open Source build platform designed for Continuous Delivery. We’re also a Continuous Delivery Foundation incubation project

You can use Screwdriver to create build pipelines for your project which is hosted in Github or Gitlab. Within a build pipeline you can configure multiple jobs to build, test and deploy your application. Explore the user guide to learn more!
